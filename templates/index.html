We've got what can help you!

The model utilizes the naive Bayes algorithm to assess the likelihood of a patient developing lung cancer based on specific risk factors. While there are many features that the user must input to utilize the model, we have used a correlation matrix to determine the top 10 features that demonstrate the highest correlation with the "Level" variable. Each of these 10 features are integers, where a negative integer is considered to be a low level of a certain feature whereas positive integer is the opposite. To generate predictions, the user must input the corresponding level of each feature, and click the submit button.
